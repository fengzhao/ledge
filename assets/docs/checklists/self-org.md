# 自组织团队建设检查清单

源自《精益产品开发：原则、方法与实践》

```checklist
 - 自组织团队检查清单
   - 信任
      - 营造相互信任的环境
   - 信息
      - 业务目标和运营状况
      - 公司及协作部门的目标和关键行动
      - 团队的目标和关键行动
      - 团队协作交付价值的过程
      - 问题、障碍和改进机会
   - 资源
      - 能力建设和培训
      - 工具、环境
      - 预算、时间
      - 获取所需信息的能力、预算和授权
   - 责任感
      - 信任、信息和资源
      - 共同认可的目标和使命
      - 即时的工作反馈
      - 即时的业务反馈
      - 正向的激励
      - 领导力
```
